Password Generator

A Python-based password generator built while learning Python fundamentals.

Features

- User chooses password length
- Generates random passwords
- Uses uppercase and lowercase letters
- Uses numbers
- Easy to extend with symbols and strength checks

Technologies Used

- Python
- Git
- GitHub

How to Run

python password_generator.py

Future Improvements

- Add symbols
- Add password strength checker
- Add copy-to-clipboard feature
- Add GUI using Tkinter
